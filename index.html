HTMX + Go Beispiel

{{if .Items}} {{len .Items}} Einträge {{else}} Noch keine Einträge {{end}}
{{range $index, $item := .Items}}
{{$item}}
{{end}}

Wie funktioniert's?

1. Text eingeben und auf "Hinzufügen" klicken

2. HTMX sendet eine POST-Anfrage an das Go-Backend

3. Go verarbeitet die Anfrage und sendet HTML zurück

4. HTMX fügt den neuen Eintrag ohne Seitenneuladung ein

5. Klicken Sie auf ✕ um Einträge zu löschen (DELETE-Anfrage)
